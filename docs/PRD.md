# SubTracker MVP PRD

## 핵심 정보

**목적**: 여러 곳에 흩어진 정기 구독 서비스를 한 곳에서 통합 관리하고 불필요한 지출을 방지
**사용자**: 개인 또는 가족/친구와 OTT, 클라우드, SaaS 구독을 공유하며 비용을 관리하는 사용자

---

## 사용자 여정

```
1. 랜딩 페이지
   ↓ [시작하기 / 로그인 버튼 클릭]

2. 로그인 페이지 / 회원가입 페이지
   ↓ [인증 성공]

3. 대시보드 페이지 (메인)
   ↓ [사용자 선택]

   [구독 추가] → 구독 추가 페이지 → 저장 완료 → 대시보드 페이지
   [구독 항목 클릭] → 구독 상세 페이지 → 수정/삭제/영수증 관리
   [그룹 메뉴] → 공유 그룹 목록 페이지 → 그룹 상세 페이지 (실시간)
   [분석 메뉴] → 지출 분석 페이지
   ↓

4. 구독 상세 페이지
   ↓ [영수증 업로드 / 수정 / 삭제]
   → 수정 완료 → 대시보드 페이지
   → 삭제 완료 → 대시보드 페이지

5. 공유 그룹 상세 페이지
   ↓ [초대 링크 공유 / 결제 상태 업데이트]
   → 실시간 반영 (모든 멤버 화면 동기화)

6. 완료 → [대시보드로 이동] 또는 [추가 구독 등록]
```

## 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F001** | 구독 CRUD | 구독 서비스 추가, 수정, 삭제 | 서비스의 핵심 가치 제공 | 구독 추가 페이지, 구독 상세 페이지, 대시보드 페이지 |
| **F002** | 구독 목록 대시보드 | 요약 카드 + 카테고리 필터 + 목록 테이블 | 전체 구독 현황 파악 | 대시보드 페이지 |
| **F003** | 결제 임박 표시 | 결제일 3일 이내 항목 빨간색 배지 표시 | 갱신 누락 방지를 위한 핵심 기능 | 대시보드 페이지 |
| **F004** | 영수증 스토리지 | 구독별 영수증/인보이스 파일 업로드, 조회, 다운로드 | 비용 증빙 관리 | 구독 상세 페이지 |
| **F005** | 공유 그룹 관리 | 그룹 생성, 초대 링크/이메일로 멤버 추가, 분담 금액 표시 | 공유 구독 비용 분담 관리 | 공유 그룹 목록 페이지, 공유 그룹 상세 페이지 |
| **F006** | 실시간 결제 상태 동기화 | 공유 구독 결제 상태 업데이트 시 모든 참여자 화면 실시간 반영 | 공유 그룹의 핵심 가치 | 공유 그룹 상세 페이지 |
| **F007** | 지출 분석 | 카테고리별 월 지출 통계 및 월별 트렌드 차트 | 구독 비용 최적화 의사결정 지원 | 지출 분석 페이지 |
| **F008** | 결제 임박 이메일 알림 | 결제일 3일 전 이메일 자동 발송 (Edge Functions + pg_cron) | 자동화된 갱신 알림 | 없음 (백그라운드 실행) |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F010** | 기본 인증 | 이메일 회원가입, 로그인, 로그아웃 | 사용자별 데이터 격리를 위한 최소 인증 | 로그인 페이지, 회원가입 페이지 |
| **F011** | 서비스 로고 업로드 | 구독 서비스 로고/이미지 업로드 (Supabase Storage) | 구독 목록 시각적 식별 | 구독 추가 페이지, 구독 상세 페이지 |

### 3. MVP 이후 기능 (제외)

- 모바일 앱
- 은행 계좌 연동 및 자동 구독 감지
- 프로필 상세 관리 (아바타, 자기소개)
- 알림 채널 설정 (푸시, SMS)
- 소셜 로그인 (Google, GitHub 등)
- 고급 검색 및 필터링

```
└── 공유 그룹 상세 - F005, F006
    └── 기능: 그룹 구독 목록, 실시간 결제 상태, 멤버 분담 금액, 초대 링크

공통 메뉴 (로그인 후)
└── 로그아웃
```

## 기술 스택 (최신 버전)

### 프론트엔드 프레임워크

- **Next.js 15.5.3** (App Router) - React 풀스택 프레임워크, 서버 컴포넌트 우선
- **TypeScript 5.6+** - 타입 안전성 보장
- **React 19** - UI 라이브러리 (최신 동시성 기능)

### 스타일링 & UI

- **TailwindCSS v4** (설정파일 없는 새로운 엔진) - 유틸리티 CSS 프레임워크
- **shadcn/ui** - 고품질 React 컴포넌트 라이브러리
- **Lucide React** - 아이콘 라이브러리

### 폼 & 검증

- **React Hook Form 7.x** - 폼 상태 관리
- **Zod** - 스키마 검증 라이브러리

### 백엔드 & 데이터베이스

- **Supabase** - BaaS 통합 플랫폼
  - PostgreSQL - 관계형 데이터베이스 + RLS
  - Auth - 이메일 기반 사용자 인증
  - Storage - 로고 및 영수증 파일 저장
  - Realtime - 공유 그룹 결제 상태 실시간 동기화
  - Edge Functions + pg_cron - 결제 임박 이메일 알림 자동화

### 이메일

- **Resend** - 결제 임박 이메일 알림 발송 (추후 변경 가능)

### 차트

- **Recharts** - 지출 분석 차트 (카테고리별 바 차트, 월별 라인 차트)

### 배포 & 호스팅

- **Vercel** - Next.js 15 최적화 배포 플랫폼

### 패키지 관리

- **npm** - 의존성 관리
